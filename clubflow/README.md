# ClubFlow
## Gestão de Clubes Esportivos

**Stack:** React + Supabase + Vercel

---

## 🚀 Quick Start

```bash
git clone https://github.com/Bridge-tecnologia-e-consultoria/clubflow.git
cd clubflow
npm install
npm run dev
```

## 🧪 Testing

```bash
npm test -- --coverage
```

## 📦 CI/CD

### Workflows
- `ci-cd.yml` - Build & deploy
- `security.yml` - Security scanning
- `coverage.yml` - Coverage enforcement

## 🚢 Deployment

- **Staging:** Push to develop
- **Production:** Push to main
